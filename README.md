# 🀄 麻将学习坊 (Mahjong Learning Workshop)

一个零基础学麻将的交互式网页应用，支持 **8 种语言**，现已加入 **完整对局模式**！

## 功能模块

- **🀫 牌型认知** — 34 种麻将牌可视化图鉴
- **📖 规则讲解** — 从牌数构成到番型简介，图文并茂
- **🧩 手牌搭建** — 交互式组牌 + 内置胡牌判定引擎
- **🎯 趣味测验** — 10 道麻将知识题
- **🎮 对局模式** — 4 人标准麻将！你 vs 3 个 AI，摸牌、打牌、碰吃杠胡，完整对局体验

## 支持语言

中文 · English · Français · Italiano · Español · Deutsch · 한국어 · 日本語

## 使用方式

在线体验：https://shengjiang-yaya11.github.io/mahjong-learn/

或下载 `index.html` 双击用浏览器打开。
